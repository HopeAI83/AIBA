Before submitting the GitHub repository link to the journal:

- [ ] Repository is public.
- [ ] `README.md` is visible and renders correctly.
- [ ] Complete Python code is present under `src/aiba_ehr/`.
- [ ] `data/synthetic_ehr_access_events_120000.csv` is present.
- [ ] `data/data_dictionary.csv` is present.
- [ ] `data/cv_fold_assignments.csv` is present.
- [ ] `data/SHA256SUMS.txt` is present.
- [ ] `results/` contains all result CSV/JSON files.
- [ ] `figures/` contains generated figures.
- [ ] `scripts/reproduce_all_results.py` runs successfully.
- [ ] `scripts/verify_repository_readiness.py` passes.
- [ ] `docs/MANUSCRIPT_AVAILABILITY_STATEMENTS.tex` has been updated with the final GitHub URL.
- [ ] The manuscript Data Availability and Code Availability statements use the same GitHub URL.
- [ ] The response letter includes the same GitHub URL.
- [ ] No DOI/Zenodo/Figshare claims remain in the manuscript or response.
